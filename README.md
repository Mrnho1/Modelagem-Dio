# E-Commerce Database

Este projeto contém o modelo e o script SQL de um banco de dados para um sistema de **e-commerce**, incluindo entidades como clientes, produtos, vendedores e o relacionamento entre eles.

---

## Estrutura do Banco de Dados

O banco possui as seguintes tabelas:

### `clients` `product` `orders` `payments` `productOrder` `productStorage` `supplier` 
### `seller` `productSeller` `storageLocation` `productSupplier` `delivery`

## Tecnologias Utilizadas

- SQL
- MySQL Workbench

---

## Como Executar

1. Clone este repositório
2. Abra o MySQL Workbench
3. Execute o script de criação das tabelas
4. Execute os inserts
5. Utilize o banco para testes e consultas
